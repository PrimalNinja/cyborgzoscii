What This Does

This application creates information-theoretically secure copies of your files. Once secured, the files are provably unbreakable - even with unlimited computing power.

Store secured files anywhere: Cloud storage, USB drives, external drives, shared servers - anywhere without worry.

How It Works

  1. Keep originals on your PC - Work with files normally
  2. Process files - Creates secured versions
  3. Store secured files externally - Cloud, USB, external storage, anywhere
  4. Keep ROM files safe - You need these to access secured data later
That's it! Your secured files can be stolen, lost, or accessed by others - they remain completely secure.

ROM Files - CRITICAL ⚠️

⚠️ WARNING: LOSE YOUR ROM FILE = LOSE YOUR DATA FOREVER ⚠️

Your ROM file is the ONLY key to access secured data. There is NO recovery method. NO backup. NO way to retrieve your data if you lose all ROM file copies.

You MUST:

  • Backup ROM files multiple times - Minimum 3 copies on different detachable/removable drives (not the same crappy/unreliable brand)
  • Store in different physical locations - Home, work, safe deposit box, trusted family
  • NEVER secure ROM files - Must stay accessible to access your secured data
  • Test backups regularly - Verify they actually work before you need them
If you lose all ROM file copies, your secured data is permanently destroyed. No exceptions.

Perfect For

  • Cloud backups - Dropbox, Google Drive, OneDrive, etc.
  • External storage - USB drives, external HDDs/SSDs
  • Off-site backups - Safe from fire, flood, theft
  • Shared storage - Where others might have access
  • Untrusted locations - Any storage you don't fully control

Common Questions

What if my USB/cloud storage is stolen?
No problem - secured files are safe without ROM files.

What if I lose my ROM file?
Your data is gone forever. No recovery possible. Keep multiple backups!

Can I share secured files?
Yes - they're safe to share. But only share ROM files with people you completely trust.

Do originals stay on my PC?
Yes - this creates secure copies for external storage. Your originals remain for normal use.
